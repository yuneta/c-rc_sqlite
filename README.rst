Yuneta rc_sqlite
================

Resources implemented with sqlite

Libraries needed by yuneta-rc_sqlite.
Add to your project:

    /yuneta/development/output/lib/libsqlite3.a
    dl          # used by sqlite
    m           # used by sqlite
